# 19-verda — Sage Botanical System

## Visual Thesis

A sage-dominant botanical system with natural structure and premium calm.

## Colour Thesis

Sage is the main structural colour across backgrounds, bands, routes, and dividers. Ivory is breathing space. Bronze/gold is botanical detail. Blush is minimal.

## Systems

- Header: `route-map-header`
- Mobile: `studio-card`
- Footer: `warm-reassurance`
- Homepage anatomy: `technical-table`
- Internal page anatomy: `softline-sequence`
- Component strategy: atelier-frame with dark-anchor calls to action
- CTA strategy: whatsapp-strip
- Image strategy: Selective full transparent portraits in sections 1, 5, and 8; object-fit contain only.

## Homepage Colour Sequence

1. Sage structural hero
2. Ivory values clearing
3. Sage route system
4. Cream consultation path
5. Sage-deep reassurance
6. Sage-soft education
7. Ivory results note
8. Cream journal path
9. Sage contact field
10. Deep sage botanical footer

## All Other Pages

Use sage as page architecture, not just decoration.

## Forbidden Patterns

- No mostly-ivory minimalism
- No dark luxury
- No blush-heavy warmth.
