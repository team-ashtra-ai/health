# Archive Plan

Date: 2026-06-26

## Proposed Archive Root

Use:

```txt
_archive/refactor-cleanup/2026-06-26/
```

Do not create or move files into this folder until human approval.

## Archive Principles

- Archive before delete whenever the file has historical, audit, screenshot, script, planning, brand, content or legal value.
- Keep archive moves grouped by reason and date.
- Preserve relative paths inside the archive so a file can be restored without guessing where it came from.
- Add a manifest before any move.
- Run verification before and after any archive batch.

## Proposed Archive Structure

```txt
_archive/refactor-cleanup/2026-06-26/
  MANIFEST.md
  audit-screenshots/
  final-homepage-screenshots/
  superseded-audit-reports/
  superseded-agent-docs/
  generated-translation-output/
  obsolete-script-cache/
  investigation/
```

## Archive Sequence

1. Create a branch for cleanup only.
2. Generate current screenshots and audits after any approved POC refactor.
3. Compare old and new screenshot/contact-sheet coverage.
4. Write `MANIFEST.md` with original path, archive path, reason, verification and rollback.
5. Archive screenshot/report outputs that are superseded.
6. Archive generated bytecode cache only if deletion is not approved yet.
7. Keep source files in place unless a later implementation plan explicitly replaces them.

## Never Archive Without Separate Approval

- `concepts/**`
- `assets/**`
- `brand identity/**`
- `data/brand.json`
- `data/concepts.json`
- `data/treatments.json`
- `docs/sofiati-master-brief.md`
- `docs/current-task-brief.md`
- `docs/sofiati-task-ledger.md`
- `AGENTS.md`
- `sitemap.xml`
- `robots.txt`
- any legal, privacy, cookie, accessibility, treatment or testimonial copy source

## Verification Commands Before Archive

```bash
git status --short --branch
find scripts -type f -name '*.pyc' -print
python3 scripts/audit_static_site.py
python3 scripts/audit_internal_links.py
python3 scripts/audit_layout_signatures.py
python3 scripts/audit_ethics.py
```

Run screenshot regeneration only when the archive batch involves screenshots, because screenshot scripts overwrite output folders.
